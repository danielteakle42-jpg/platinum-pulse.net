PLATINUM PULSE PORTAL

How to run:
1. Open this folder in VS Code
2. Run:
   npm install
   npm run dev
3. Open:
   http://localhost:3000

Included routes:
- /
- /leaderboard
- /incentives
- /admin
- /dashboard/chaossprout
